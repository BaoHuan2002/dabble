To run this project, please follow these steps:

1) Run database migrations

  + Go to the app-migrations folder.

  + Run the main file to execute database migrations.

2) Run the backend server

  + Go to the server folder.
  
  + Run the DabbleApplication file to start the backend service.

3) Run the frontend
  + Go to the frontend folder.
  + npm install
  + npm start
